# E-Commerce Data Analysis Dashboard 🛒

## Setup Environment - Shell/Terminal

Buka terminal pada folder `dashboard`, lalu install library yang dibutuhkan:

```bash
python -m pip install streamlit pandas matplotlib seaborn
```

---

## Open Integrated Terminal

- Klik kanan pada folder `dashboard`
- Pilih **Open Integrated Terminal**

---

## Run Streamlit App

Jalankan dashboard menggunakan perintah berikut:

```bash
python -m streamlit run dashboard.py
```

Dashboard akan otomatis terbuka pada browser lokal.

---

## Generate requirements.txt

Buka terminal pada folder `Submission`, lalu jalankan:

```bash
python -m pip freeze > requirements.txt
```
